# Audio Asset Policy

CPUpovic can only bundle audio that is safe to redistribute.

## Allowed

- CC0 audio
- Public domain audio
- Audio with an explicit redistributable license compatible with this repository
- Project-maintainer-created AI-generated sound effects when the generation tool terms allow project use and the asset notice documents provenance separately from the code license

## Allowed With Attribution

- Compatible CC-BY audio, if attribution is committed beside the asset
- Other compatible licenses that require attribution, if the attribution text is complete and clear

## Avoid For Bundled Defaults

- Unclear "royalty-free" marketplace assets where standalone redistribution is not allowed
- Assets that require account-specific license grants
- Assets whose license cannot be verified from a public source URL

## Forbidden

- YouTube, TikTok, Instagram, or social media rips
- Branded Akrapovic, Ferrari, Lamborghini, F1, or other vehicle/exhaust samples
- Game, film, broadcast, or commercial music audio
- Any unclear-license sound

## Required Metadata

Every bundled audio file needs an entry in the soundpack `LICENSES.md`:

- Local filename
- Source URL or generation tool/session note
- Author or creator
- Generation tool or source
- License or asset terms
- Download date, generation date, or documented asset date
- Notes on redistribution compatibility

If that metadata is missing, do not commit the audio file.

## Local Engine Pack

The current Local Engine Pack uses fictional Adobe Firefly Sound Effects generated by the project maintainer. These files are documented in `public/soundpacks/local-engine/LICENSES.md`.

The repository MIT license covers project code. Audio asset terms are documented separately and should not be described as MIT-licensed code.

## Supara Pack

The Supara Pack uses fictional Adobe Firefly Sound Effects generated by the project maintainer. These files are documented in `public/soundpacks/supara/LICENSES.md`.

The pack is not affiliated with Toyota, Supra, Akrapovic, Ferrari, Lamborghini, Formula 1, or any vehicle/exhaust brand. It must not be described as a real branded recording.
