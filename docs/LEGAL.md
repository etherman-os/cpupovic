# Legal Notes

CPUpovic is not affiliated with any vehicle manufacturer, racing organization, or exhaust company.

## Repository Rules

- No real exhaust branding, logos, or product names.
- No vehicle brand logos or model names for bundled profiles.
- No copyrighted engine or exhaust samples.
- No extracted audio from videos, games, films, or broadcasts.
- Bundled audio must be synthetic, public domain, explicitly safe to redistribute, or documented as project-maintainer-created/generated assets with separate asset terms.
- Compatible CC-BY assets need attribution committed beside the audio files.
- Every bundled audio file needs source/generation details, author or creator, asset terms, date if known, and local filename metadata.

Users may experiment locally with their own soundpacks, but those assets should not be committed unless their license is clear and compatible.

## Code License And Audio Assets

The CPUpovic source code is licensed under the repository's MIT license.

Audio assets are documented separately from the code license. Do not describe bundled `.wav` files as MIT-licensed code unless a separate asset notice explicitly says so.

## Why CPUpovic Uses Samples/Hybrid Audio

Synthetic audio is legally simple, but it can sound too artificial. CPUpovic supports a sample-hybrid path so safe redistributable loop layers can provide better engine-like character. The legal rule is stricter than the technical rule: if provenance is unclear, the file does not belong in the repo.

## Local Engine Pack

The Local Engine Pack audio files are fictional AI-generated sound effects created by the project maintainer with Adobe Firefly Sound Effects.

They are not recordings of Akrapovic, any branded exhaust, any vehicle manufacturer, racing series, game, film, broadcast, YouTube, TikTok, Instagram, or social media clip.

The app can use these sounds as the default Local Engine Pack. Their provenance is documented in `public/soundpacks/local-engine/LICENSES.md`.
