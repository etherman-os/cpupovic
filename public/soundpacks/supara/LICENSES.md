# Supara Pack Licenses

The Supara Pack audio files were generated with Adobe Firefly Sound Effects by the project maintainer.

These are fictional AI-generated turbo tuner-style engine sounds. They are not recordings of Toyota Supra, Akrapovic, Ferrari, Lamborghini, Formula 1, any real branded vehicle, or any real branded exhaust system.

## Code License vs Audio Notes

The CPUpovic source code is licensed separately under the repository's MIT license.

These audio files are assets, not source code, and are not described as MIT-licensed code. Their generation notes and usage context are documented separately so CPUpovic can use them as a user-facing sound pack.
