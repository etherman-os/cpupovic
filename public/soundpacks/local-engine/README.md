# Local Engine Pack

This folder contains the default Local Engine Pack for CPUpovic.

Minimum required files:

- `idle.wav`
- `low.wav`
- `mid.wav`
- `high.wav`

Optional files:

- `rev_burst.wav`
- `cooldown.wav`

The app treats this pack as ready only when all four required loop files are present and decodable. Optional one-shots can be added later.
